# Claude Code Instructions

This repo's canonical agent instructions live in [`AGENTS.md`](./AGENTS.md). Read it first — it is the single source of repo rules so that Claude Code and Codex load identical guidance without duplication.

Governance is split across three documents (the three-document model): the [Process Doc](./gentech_slice_based_development_process_revised.md) holds the rules, the [Skill Usage Guide](./gentech_agent_skills_usage_and_requirements.md) is the skill index, and the [Orchestration Map](./gentech_slice_lifecycle_orchestration_map.md) is the control flow. `AGENTS.md` points into all three.
